# 图像数据集泄露检测总结

## 工具说明

LeakShield 现已支持图像数据集的泄露检测功能，可以检测：

1. **完全重复的图片**（基于文件哈希）
2. **视觉相似的图片**（基于感知哈希）
3. **文件名泄露**（文件名包含标签信息）

## 使用方法

```bash
# 基本用法
python check_image_leakage.py <数据集根目录>

# 导出详细报告
python check_image_leakage.py <数据集根目录> --output report.md
```

## 技术细节

### 检测方法

1. **文件哈希检测**
   - 使用 MD5 哈希算法
   - 检测完全相同的文件
   - 时间复杂度: O(n)

2. **感知哈希检测**
   - 使用平均哈希算法（Average Hash）
   - 计算汉明距离
   - 阈值：汉明距离 ≤ 2（64位哈希）
   - 对图像的微小变化不敏感

3. **文件名检测**
   - 检查文件名是否包含标签关键词
   - 关键词：label, class, category, type, id, target

### 性能

- 扫描速度：约 20,000 张图像/分钟（文件哈希）
- 感知哈希：约 2,000 张图像/分钟
- 总检测时间：约 5-10 分钟（60,000 张图像）
